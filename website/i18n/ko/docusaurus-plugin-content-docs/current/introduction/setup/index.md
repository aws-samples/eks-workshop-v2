이 섹션에서는 워크샵의 실습을 실행하기 위한 환경 설정 방법을 설명합니다.

워크샵 환경에 액세스하는 두 가지 옵션이 있습니다:

1. [AWS 이벤트에서](./aws-event.md) - AWS 주최 워크샵 이벤트에 참석하는 참가자용
2. [본인 계정에서](./your-account/index.md) - 개인 AWS 계정에서 워크샵을 실행하는 경우

:::info
각 옵션은 워크샵 실습을 완료하는 데 필요한 리소스 및 도구에 대한 액세스 권한을 얻기 위한 자세한 지침을 제공합니다.
:::

귀하의 상황에 가장 적합한 옵션을 선택하여 환경 설정을 진행하세요.
