이 워크샵에 참여하시면 실습 자료를 완료하는 데 사용할 AWS 계정이 제공됩니다. [https://catalog.workshops.aws/](https://catalog.workshops.aws/)로 이동하여 포털에 연결합니다. **Get Started**를 클릭합니다.

![Workshop Studio Home](/docs/introduction/setup/workshop-studio-home.webp)

로그인하라는 메시지가 표시됩니다. **Email One-Time Password(OTP)** 옵션을 선택합니다.

![Workshop Studio Sign in](/docs/introduction/setup/ws-studio-login.webp)

이메일 주소를 입력하고 **Send passcode**를 누르면 일회용 비밀번호가 이메일로 전송됩니다. 이메일이 도착하면 비밀번호를 입력하고 로그인합니다.

강사가 이 실습을 시작하기 전에 **Event access code**를 제공했을 것입니다. 제공된 코드를 텍스트 상자에 입력하고 **Next**를 클릭합니다.

![Event Code](/docs/introduction/setup/event-code.webp)

이용 약관을 읽고 동의한 후 **Join event**를 클릭하여 계속 진행합니다.

![Review and Join](/docs/introduction/setup/review-and-join.webp)

개인 대시보드가 표시됩니다. **Open AWS Console** 버튼을 선택하여 AWS 계정 콘솔로 이동합니다:

![Open Console](/docs/introduction/setup/openconsole.webp)

다음으로 개인 대시보드 페이지로 돌아가서 **Event Outputs** 섹션까지 아래로 스크롤합니다. **IdeUrl** 필드에서 URL을 복사하여 새 브라우저 탭에서 엽니다:

![Cloud9 Link](/docs/introduction/setup/workshop-studio-06.png)

비밀번호를 입력하라는 메시지가 표시됩니다:

![IDE Password](/docs/introduction/setup/visual-studio-01.png)

출력에서 **IdePassword** 필드의 값을 입력하면 웹 IDE가 로드됩니다.

![Code-server login screen](/docs/introduction/setup/vscode-splash.webp)

**Get started**를 눌러 워크샵 시작 페이지에 액세스합니다:

![Get Started](/docs/introduction/setup/workshop-event-page.webp)

이제 [실습 탐색 섹션](/docs/introduction/navigating-labs)으로 진행할 수 있습니다.
