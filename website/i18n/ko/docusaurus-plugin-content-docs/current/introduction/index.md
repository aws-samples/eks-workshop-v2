**AWS Elastic Kubernetes Service (EKS) 워크샵**에 오신 것을 환영합니다!

이 워크샵은 EKS에서 제공하는 다양한 기능과 AWS에서 제공하는 광범위한 서비스들과의 통합 방법을 학습하고 탐색하기 위한 실습 랩 연습 세트를 안내합니다. 랩은 다음과 같은 여러 영역으로 그룹화되어 있습니다:

- **소개** - 이 워크샵의 형식과 구조를 학습합니다
- **기본 사항** - 관리형 노드 그룹, Fargate, 애플리케이션 노출 및 스토리지 활용과 같은 기본적인 EKS 개념에 익숙해집니다
- **오토스케일링** - 애플리케이션과 클러스터를 수평 및 수직으로 자동으로 확장하는 방법을 이해합니다
- **관측 가능성** - 모니터링은 워크로드를 프로덕션으로 전환하는 데 중요한 요소입니다
- **보안** - 컨테이너 보안 관행을 EKS 클러스터와 워크로드에 적용하는 방법을 시연합니다
- **네트워킹** - EKS 네트워킹이 Amazon Virtual Private Cloud 등과 통합되는 방식을 심층적으로 살펴봅니다
- **자동화** - GitOps 및 EKS를 통한 인프라 프로비저닝과 같은 원칙을 적용합니다

이러한 랩은 여러분의 경험과 기술을 반영하는 모듈의 조합을 선택하여 효율적으로 학습할 수 있도록 설계되었습니다. 워크샵 환경은 EKS 클러스터를 포함한 사전 구성된 AWS 인프라와 랩 연습을 완료하는 데 필요한 모든 도구를 갖춘 통합 개발 환경(IDE)을 제공합니다.
