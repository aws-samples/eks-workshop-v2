조직에서 AI 및 ML 기술을 점점 더 많이 채택함에 따라, 확장 가능하고 효율적인 인프라의 필요성이 중요해지고 있습니다. Amazon EKS는 ML 워크로드를 배포하고 관리하기 위한 강력한 플랫폼을 제공하며, Kubernetes의 유연성과 특수한 AWS ML 액셀러레이터 및 서비스의 원활한 통합을 제공합니다. 이 실습에서는 ML 모델 학습 및 배포를 위해 EKS를 활용하는 방법, GPU 리소스 활용을 최적화하는 방법, 대규모 ML 파이프라인 오케스트레이션을 위한 모범 사례 구현, 그리고 자연어 명령을 사용하여 EKS 클러스터 작업을 수행하기 위해 Amazon Q CLI를 사용하는 방법을 배우게 됩니다. 실시간 추론, 분산 학습 또는 생성형 AI 워크로드 작업 중 어떤 것이든, EKS는 프로덕션 수준의 ML 운영에 필요한 견고한 기반을 제공합니다.
