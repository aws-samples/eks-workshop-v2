현재 웹 스토어 애플리케이션은 외부 세계에 노출되어 있지 않으므로 사용자가 접근할 방법이 없습니다. 웹 스토어 워크로드에는 많은 마이크로서비스가 있지만, 최종 사용자에게 제공되어야 하는 것은 `ui` 애플리케이션뿐입니다. 이는 `ui` 애플리케이션이 내부 Kubernetes 네트워킹을 사용하여 다른 백엔드 서비스와의 모든 통신을 수행하기 때문입니다.

이 워크샵 챕터에서는 EKS를 사용할 때 최종 사용자에게 애플리케이션을 노출하는 다양한 메커니즘을 살펴보겠습니다.
