Kubernetes 네트워킹을 이해하는 것은 클러스터와 애플리케이션을 효율적으로 운영하는 데 매우 중요합니다. 이 장에서는 Pod 네트워킹, 서비스 네트워킹, 서비스 통신을 포함한 Kubernetes 네트워킹의 다양한 측면에 대해 살펴보겠습니다.

Amazon EKS에서 Pod 네트워킹은 클러스터 네트워킹이라고도 하며, Amazon VPC CNI라는 Kubernetes CNI 플러그인을 통해 구현됩니다. Amazon VPC Lattice로 넘어가기 전에 Amazon VPC CNI에서 사용할 수 있는 다양한 옵션을 살펴보는 것을 적극 권장합니다.
